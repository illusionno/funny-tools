### DOM元素查看
类似于开发者工具中的元素查看, 但是只能查看元素的DOM结构，可以对元素节点进行展开和折叠。
实现过程：使用[html-dom-parser](https://github.com/remarkablemark/html-dom-parser)库解析html，然后使用递归遍历 JavaScript 对象，生成嵌套的节点结构。
